# Retro Pong 2084

Juego estilo Pong en consola con múltiples modos: solitario, IA, multijugador.  
Incluye ranking persistente, animaciones en colores y sonidos básicos.  

## Cómo ejecutar
1. Abrir en Delphi o Free Pascal.
2. Compilar `juegocompleto_final.pas`.
3. Ejecutar el binario resultante.

## Controles
- Jugador1: W/S
- Jugador2: P/L
- Pausa: M
- Salir: ESC
